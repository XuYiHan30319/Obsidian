# 函数模板

[toc]

## template\<class T\>

我们可以设置函数模板函数

```c++
template<class T>
void swapVal(T &a,T&b);...
```



模板函数如果不能满足要求,可以自己再定一个新的用于替换

在存在了swapVal(T)外,可以自己定义一个swapVal(int)

如果有了同名的具体实现函数,那么模板不会生成
