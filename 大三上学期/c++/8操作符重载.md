# 类的类型转换
<font color="#ff0000">转换构造函数</font>.在c++中只有一个变量的函数就是转换构造函数
```c++
class Stock{
public:
	Stock(int a);
	Stock(double a,int a=0);//也可以
}

Stock a;
a = 1;//可以的,首先构造一个临时的stock(1)再拷贝给stock
```
当初始化,赋值,传参或者返回的时候会发生隐式类型转换.如果不希望发生类型转换,可以使用explicit Stock(int a);
