```c++
class classname{
private:
	...
public:
	classname(int a = 0);
	void a()const;
protected:
	...
};

void classname::void a()const;//都要加
classname::classname(int a){...}//只要有一处标记就行
```
c++的类默认为private的,并且定义在类内的函数默认是inline的,inline放哪都行,都可以过编译,但是具体是啥得看实际情况
