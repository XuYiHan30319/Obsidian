## 共同体类型union
```c++
struct widget{
	int type;
	union{
		long id_num;
		char id_char[20];
	}
}
//union就是俩变量存在一个空间,空间大小为大的那个
```

## enum
```c++
enum c{red,blue,red};//都是常量0-2
int x = blue;//x=1
```

## const+指针
```c++
const int *p
int const *p//这俩效果一样,都是用来修饰int
int * const p;//常量指针,指向不能变

void a(const int *a){
	//只读模式
}
```
